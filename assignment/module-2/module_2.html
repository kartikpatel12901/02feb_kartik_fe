Q1. What are the benefits of using CSS?

Ans.

benefits of using css are:

  1. Faster Page Speed:
    2. More code means slower page speed. And CSS enables you to use less code. CSS allows you to use one CSS rule and apply it to all occurrences of a certain tag within an HTML document.
  2. Better User Experience:
    3. CSS not only makes web pages easy on the eye, it also allows for user-friendly formatting. When buttons and text are in logical places and well organized, user experience improves.
  3. Quicker Development Time:
    4. With CSS, you can apply specific formatting rules and styles to multiple pages with one string of code. One css can be replicated across several website pages.
  4. Easy Formatting Changes:
    5. If you need to change the format of a specific set of pages, it's easy to do so with CSS. There's no need to fix every individual page. Just edit the corresponding CSS stylesheet and you'll see changes applied to all the pages that are using that style sheet.
  5. Compatibility Across Devices:
    6. Responsive web design matters. In today's day and age, web pages must be fully visible and easily navigable on all devices. Whether mobile or tablet, desktop, or even smart TV, CSS combines with HTML to make responsive design possible.

Q2. What are the disadvantages of CSS?

Ans.

Disadvantages of css are:

  1. CSS, CSS 1 up to CSS3, result in creating of confusion among web browsers.
  2. With CSS, what works with one browser might not always work with another. The web developers need to test for compatibility, running the program across multiple browsers.
  3. There exists a scarcity of security.
  4. CSS data is interpreted depending on the internet browser version. If the visitors are using an outdated version, they won't be able to see the design properly.

Q5. What do you understand by CSS opacity?

Ans.

Opacity property is used to specify the transparency of an element.

The opacity property can take a value from 0.0 - 1.0. The lower the value, the more transparent:

Ex.

Opacity in div :-
Syntex: .class_name{background: 127, 255, 212; opacity:0.4;}
Opacity in background:-
Syntex: .class_name{background-color: rgba(127, 255, 212, 0.4);}

Q9. Which property controls the image scroll in the background?

Ans.The background-attachment property in CSS is used to specify the kind of attachment of the background image with respect to its container. It can be set to scroll or make it remain fixed.

Syntax: .class_name{background-attachment: scroll}
