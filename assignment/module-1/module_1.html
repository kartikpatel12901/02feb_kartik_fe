Q10. How can we club two or more rows or columns into a single row or column in an HTML table?

Ans. We uses “rowspan” and “colspan” attribute to merge cells in HTML.The rowspan attribute is for the number of rows a cell should merge, whereas the colspan attribute is for the number of columns a cell should merge.

Syntax of rowspan:- <td rowspan=“3”> content </td>

Syntax of colspan:- <td colspan=“3”> content </td>
